# Astrology Engine

Supported systems:

- Tropical
- Sidereal
- Natal Chart
- Solar Return
- Progressions
- Secondary Progressions
- Synastry
- Composite
- Astrocartography
- Electional Astrology
- Lunar Returns
- Transits

Astrology is used as one layer of personalized reflection and planning.

The system combines astrological information with user goals, context and evidence-based planning.

The application must never present astrology as guaranteed prediction.

It should always explain uncertainty clearly.
