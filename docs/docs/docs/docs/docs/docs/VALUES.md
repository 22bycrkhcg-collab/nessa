# NÉSSA Core Values

## 1. Truth over hype

We value evidence, transparency and honest uncertainty.

## 2. Human first

Technology should empower people, not replace them.

## 3. Excellence

Every feature should feel premium.

## 4. Long-term thinking

We optimize for decades, not weeks.

## 5. Privacy

User data belongs to the user.

## 6. Continuous growth

Small improvements every day create extraordinary results.

## 7. Simplicity

Complex intelligence.
Simple experience.
